# voice-note Editor Link Report

## Summary

Added editor-aware transcript opening for the TUI.

## Changes

- Added `--editor code|nvim`.
- Added `editor` setting with default `code`.
- Changed VS Code transcript links to `vscode://file/...`.
- Kept nvim transcript links as `file://...` and added an `o` binding that
  suspends the TUI and runs `nvim transcribe.txt`.
- Added helper tests for editor normalization and transcript URLs.
- Updated README and implementation plan.

## Checks

- `PYTHONPATH=src/voice-note uv run python -m compileall src/voice-note/voice_note src/voice-note/tests`
- `PYTHONPATH=src/voice-note uv run python -m unittest discover -s src/voice-note/tests`
- `uv run voice-note --help`
- Headless Textual checks for VS Code and nvim transcript link URLs.
