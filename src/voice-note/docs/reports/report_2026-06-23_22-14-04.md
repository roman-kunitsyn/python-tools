# voice-note TUI Session Header Report

## Summary

Replaced the `Previous Notes` TUI label with the current session name and a
terminal link to the session `transcribe.txt` file.

## Changes

- Exposed `VoiceNoteService.output_file` from the configured writer.
- Added writer `output_file` metadata.
- Added TUI session-name rendering from the transcript file parent directory.
- Added a Textual `Link` widget for the transcript path with a `file://` URL.
- Added tests for session name, transcript label, and transcript URL helpers.
- Updated README.

## Checks

- `PYTHONPATH=src/voice-note uv run python -m compileall src/voice-note/voice_note src/voice-note/tests`
- `PYTHONPATH=src/voice-note uv run python -m unittest discover -s src/voice-note/tests`
- Headless Textual `run_test()` mounted `VoiceNoteApp` and verified the
  transcript link text and URL.
