# voice-note JSON Transcript Report

## Summary

Added structured JSON transcript output and forced Whisper output to English.

## Changes

- Added default `transcribe.json` in each session folder.
- Added `--json-output-file` override.
- Added `TranscriptJsonWriter`.
- `VoiceNoteService` now writes every completed note to both `transcribe.txt`
  and `transcribe.json`.
- JSON schema:
  `{"session": "...", "data": [{"audio": "...", "text": "..."}]}`.
- Added `whisper-cli -tr` so voice-note output is translated to English.
- Restored the recording limit default to 300 seconds.
- Updated docs and tests.

## Checks

- `PYTHONPATH=src/voice-note uv run python -m compileall src/voice-note/voice_note src/voice-note/tests`
- `PYTHONPATH=src/voice-note uv run python -m unittest discover -s src/voice-note/tests`
- `uv run voice-note --help`
