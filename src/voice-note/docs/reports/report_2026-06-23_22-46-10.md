# voice-note Recording Limit Report

## Summary

Added a configurable per-recording time limit with countdown display and
automatic stop on timeout.

## Changes

- Added `max_recording_seconds` setting with default `300`.
- Added `--max-recording-seconds` CLI option.
- Validated the limit must be greater than `0` and no more than `300`.
- Passed the limit to `audio-record` / FFmpeg as the recording duration.
- Added CLI countdown while recording.
- Added TUI footer countdown while recording.
- Added automatic stop when the countdown reaches zero.
- Added overflow message: `Record Stop by time overflow`.
- Added overflow status styling.
- Updated docs and tests.
