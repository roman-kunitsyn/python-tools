## Behavior Added

- Thin CLI entry point for the browser automation tool.
- JSON-based configuration loading with command-line overrides.
- Playwright browser session abstraction.
- Scenario recording wrapper around Playwright codegen.
- Python scenario execution with browser/context/page injection.
- Recursive site crawling with de-duplication and page limits.
- Markdown, HTML, text, link, image, screenshot, and PDF export helpers.
- Example scenario template for future workflows.

- `uv run` could not be validated in this sandbox because it attempted to reach
  PyPI and network access is restricted here.
- Robots.txt, sitemap discovery, and richer failure handling remain future
